1. Project name
Final Capstone

2. Description
Jupyter notebook for unsupervised machine learning using the US arrests dataset of statistics, in arrests per 100,000 residents, for assault, murder, and rape in each of the 50 US states in 1973. The notebook develops clustering and analysis of the clusters generated using principal component analysis techniques.

3. Installation
Download UsArrests.csv and the Jupyter notebook. The Jupyter notebook requires UsArrests.csv to be saved in the same folder as the notebook.

4. Usage
Designed for use with Visual Studio Code in Python 3.11. Jupyter is required in the installation environment. Written in Jupyter v2023.1.2010391206.
